变量

  1. Python 的变量大小写敏感
  2. 字母数字下划线，不能以数字开头
  3. Python 是动态类型的语言，Python 中的变量不需要声明类型。每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。
  4. 不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）；
  5. 可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

Number

只有int，没有长整型

type（）函数可以用来查看变量类型

string

\可以用来转义，使用r可以让反斜杠不发生转义。

元组

python 的元组与列表类似，不同之处在于元组的元素不能修改。

  1. 创建元组用（），或者不用括号也可以
  2. 可以创建空元组
  3. 元组中只包含一个元素时，需要在元素后面加,不然（）会被当做运算符
  4. 可以使用下标访问元组中的值
  5. 元组不可被修改

不能被修改但是可以连接

可变数据：

列表

  1. 用[]创建
  2. 可以索引，切片，加，乘，检查成员

字典

  1. 用{}创建
  2. 字典是另一种可变容器模型，且可存储任意类型对象。

访问字典的值

集合

  1. 集合（set）是一个无序的不重复元素序列。
  2. 集合中的元素不会重复，并且可以进行交集、并集、差集等常见的集合操作。
  3. 可以使用大括号 { } 创建集合，元素之间用逗号 , 分隔， 或者也可以使用 set() 函数创建集合。

创建格式

注意：创建一个空集合必须用 set() 而不是 **{ }**，因为 { } 是用来创建一个空字典。

添加元素

  1. s.add(x)
  2. s.update(x) x可以是列表，元组，字典等

判断元素是否在集合中存在
